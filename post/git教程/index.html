Git教程

2021-01-29 Git

git init 把当前的目录变成git仓库，生成隐藏.git文件。
git status 查看仓库的改变情况，会有相关的提示操作出现

git add .把新的文件添加到本地暂存区说明

git commit -m “note” 提交到本地仓库

git remote add origin https://github.com/uuvip/uuvip.github.io.git 关联GIETT

git push 将改动上传到github，若没有指定分支，则需要使用git push origin master
git push -u origin master 推向分支
git push origin +master 强制推送
git pull 拉取代码
git pull origin master

Git首次配置
git config --global user.email "376444013@qq.com"

git config --global user.name "uu14top"

git ls-files 命令是用来查看暂存区中文件信息。
git rm --cached +文件名将文件移除暂存区
rm .git/index 清空暂存区，其实就是删除.git目录下的index文件
