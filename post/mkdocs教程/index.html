Mkdocs教程

2021-01-29
教程
约 379 字 预计阅读 1 分钟

文章目录

mkdocs 是一个静态的页面生成工具，通过python pip 安装，安装步骤如下

安装python ，pip，我安装的是python 2.7,自带有pip

这里有可能出现的是python不是可运行的命令，需要手动添加环境变量：

变量设置：控制面板\系统和安全\系统 1，点我的电脑-右键属性-高级系统设置-高级-环境变量-新建- 2，变量名：PATH 变量值：C:\Users\admin\AppData\Local\Programs\Python\Python38-32\Scripts 《变量值也就是安装python的路径后加上\Scripts》

1.安装mkdocs version

pip install mkdocs

2.创建自己的项目

mkdocs new yourProject<创建一个新文件夹> cd yourProject《CD到新建的文件夹》 一定要到创建好的目录里面启动mkdocs serve mkdocs serve《启动》 mkdocs build 《生成静态的html站点》
