docker安装web摄像头项目 motioneye

2024-04-07

docker安装web摄像头项目 motioneye
添加教程https://zhuanlan.zhihu.com/p/663587091
https://hub.docker.com/r/dontobi/motioneye.rpi
安装命令
docker run -d --name motioneye
-p 8765:8765
-e TZ="Asia/Shanghai"
-v /mnt/sda4/motioneye/motioneye:/etc/motioneye
-v /mnt/sda4/motioneye/videos:/var/lib/motioneye
-v /etc/localtime:/etc/localtime:ro
-e UID=1000
-e GID=1000
--restart=always
dontobi/motioneye.rpi:latest
注意 -v /etc/localtime:/etc/localtime:ro \不能改
（二）登录
用浏览器打开 http://容器的IP地址:8765
用户名：admin
密码：默认无，留空即可
