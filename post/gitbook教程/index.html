1,安装node.js

安装这个版本https://cdn.npm.taobao.org/dist/node/v10.15.3/node-v10.15.3-x64.msi 查看版本号 node -v v10.15.3

2,安装GitBook

npm install gitbook-cli -g

查看版本号 gitbook -V CLI version: 2.3.2 GitBook version: 3.2.3

3,初始化 先进入一个新的目录在gitbook init

gitbook init

运行 gitbook serve

命令也可以指定端口：gitbook serve –port 8088

生成的html文件 gitbook build

常用命令解析

gitbook init //初始化目录文件 gitbook help //列出gitbook所有的命令 gitbook –help //输出gitbook-cli的帮助信息 gitbook build //生成静态网页 gitbook serve //生成静态网页并运行服务器 gitbook build –gitbook=2.0.1 //生成时指定gitbook的版本, 本地没有会先下载 gitbook ls //列出本地所有的gitbook版本 gitbook ls-remote //列出远程可用的gitbook版本 gitbook fetch 标签/版本号 //安装对应的gitbook版本 gitbook update //更新到gitbook的最新版本 gitbook uninstall 2.0.1 //卸载对应的gitbook版本 gitbook build –log=debug //指定log的级别
