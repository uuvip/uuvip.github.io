ubuntu22.04服务器版WiFi连接

2022-11-12

1，先查看网卡
ip a

3: wls1: <BROADCAST,MULTICAST,UP,LOWER_UP> mtu 1500 qdisc mq state UP group default qlen 1000
第3这个wls1就是我的wifi网卡了

2，配置Netplan以连接到无线网络（SSID）
cd /etc/netplan里 看见有2个文件下面
00-installer-config-wifi.yaml 00-installer-config.yaml
编辑这个vim 00-installer-config-wifi.yaml
内容：

network:
version: 2
wifis:
wls1:
dhcp4: true
access-points:
"206":
password: "Zhang888888"

我直接通过重启服务器就OK了，也可以用一下命令。
sudo netplan generate
执行以下命令连接到无线接口
sudo netplan apply

如果一切顺利，您可以通过执行以下命令看到连接到无线网络的无线适配器：ip
$ ip a
