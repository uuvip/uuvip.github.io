n1docker安装frpc

2024-04-07

https://hub.docker.com/r/snowdreamtech/frpc
docker run --restart=always --network host -d -v /etc/frp/frpc.toml:/etc/frp/frpc.toml --name frpc snowdreamtech/frpc
修改挂载的frpc.toml配置文件如下例子：

serverAddr = "frp.104300.xyz"
serverPort = 7000
auth.method = "token"
auth.token = "frp.104300.xyz"

[[proxies]]

type = "tcp"
localIP = "192.168.1.1"
localPort = 80
remotePort = 30344
