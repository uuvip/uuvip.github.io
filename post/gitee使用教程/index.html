1，首先 下载安装

下载地址：https://git-scm.com/

下载后傻瓜式一键安装，建议安装在英文目录下

2，由于git是分布式管理工具，需要输入用户名和邮箱以作为标识，因此，我们按右键打开Git Bash命令框 在命令行输入下列的命令：git config –global user.name “班长” git config –global user.email “8280402+q376444013@user.noreply.gitee.com” name可以为中文，邮箱建议使用qq邮箱，方便记忆 3，生成SSH公钥，许多 Git 服务器都使用 SSH 公钥进行认证，所以我们也需要配置该公钥

输入如下命令，三次回车即可生成 ssh key

ssh-keygen -t rsa -C “8280402+q376444013@user.noreply.gitee.com“邮箱建议使用qq邮箱

4，常用命令

git status 查看仓库的改变情况，会有相关的提示操作出现

git add .直接添加所有改动的文件

git commit -m “note” 确认生成本地的版本，note是 版本特点说明

git remote add origin git@gitee.com:q376444013/q376444013.git 关联GIETT

git push 将改动上传到github，若没有指定分支，则需要使用git push origin master

6，远程仓库与本地仓库合并

这时候不着急推，先将远程仓库的文件同步一次到本地，否则直接推会报错 git pull –rebase origin master

执行git add . 《git add .推送全部》《git add 文件名》推送指定一个文件名 git commit -m “note” git push

8，git rm 想要删除的文件的名字及其后缀 如：git rm 1.txt
